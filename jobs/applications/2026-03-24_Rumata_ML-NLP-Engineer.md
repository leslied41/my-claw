Key responsibilities:

- Balance accuracy with inference cost and latency
- Build and maintain both LLM and classical NLP workflows
- Evaluate and benchmark algorithms systematically
- Optimize pipeline performance and cloud infrastructure costs

Required: 5+ years software engineering, 3+ years ML/NLP, PyTorch, Hugging Face, embedding models, fine-tuning, LoRA/QLoRA, RAG architectures
