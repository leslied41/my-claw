## Job Description
EnrolHQ is the leading enrolment and admissions platform for independent schools across Australia and New Zealand. We help schools modernise and streamline their entire enrolment journey – from first enquiry through to offer, acceptance, and onboarding – with powerful automation, integrations, and data workflows.

Our platform is used daily by hundreds of schools to manage tens of thousands of student applications. We're a small, focused product company where every engineer has direct impact on what we ship and how our customers experience the product.

The Role
We're hiring a Senior Developer to work on the core EnrolHQ platform. You'll own features end-to-end across a Django + Vue stack, working on everything from complex backend workflows and API integrations through to the dashboards and interfaces that school staff use every day.

This is a hands-on, high-ownership role. You'll work directly with the founders and a small engineering team to ship features, improve architecture, and solve real problems for schools. We value people who can think independently, move fast, and care about the quality of what they build.

This is not a ticket-factory role. You'll have genuine influence over technical decisions and product direction.

What You'll Do
- Build and ship full-stack features across Python/Django APIs and Vue 3/TypeScript frontends
- Design, extend, and maintain our PostgreSQL data models and optimise query performance
- Work on integrations with school SIS platforms (Sentral, Synergetic, Edumate, Veracross, Compass) and third-party services
- Build and improve async workflows using Celery, Redis, and Django Channels (WebSockets)
- Contribute to API design for our public REST API (Django REST Framework + drf-spectacular)
- Write meaningful tests (pytest, integration tests) and maintain code quality through pre-commit hooks, ruff, mypy, and ESLint
- Work with AWS infrastructure (S3, Lambda, ECS) in a production SaaS environment
- Collaborate directly with founders and the team on product and architecture decisions

Our Stack
- Backend: Python 3.13, Django, Django REST Framework, Celery, Django Channels, Pydantic
- Frontend: Vue 3, TypeScript, Vite, Bootstrap
- Data: PostgreSQL 15, Redis
- Infrastructure: AWS (S3, Lambda, ECS, RDS, CloudFront), Docker, Nix
- Quality: pytest, Jest, mypy, ruff, ESLint, pre-commit, GitHub Actions
- Tools: GitHub, Sentry, drf-spectacular (OpenAPI), MkDocs

Requirements
- 5-7 years of professional software engineering experience
- Strong backend experience with Python in production, including APIs, background jobs, and data modelling
- Solid JavaScript/TypeScript skills with experience in a modern frontend framework (Vue preferred; React or Angular also valued)
- PostgreSQL – comfortable writing and optimising queries beyond the ORM
- Experience designing and building RESTful APIs and integrating with third-party systems
- Strong testing discipline – you write tests because you want your code to work, not because someone told you to
- Ability to self-direct, investigate, and resolve issues without handholding
- Must have the right to work in Australia

Nice to Have
- Experience with Celery, Redis, WebSockets (Django Channels)
- Python type-checking experience (mypy)
- AWS experience running production systems
- Experience with or strong interest in building with AI/LLMs to improve development workflows
- Experience in education technology, SaaS, or school administration systems
- Familiarity with Vue 3 Composition API and migration from Vue 2
- Startup or small-team experience with high ownership and autonomy

How We Work
- Hybrid work model - office in Ultimo + work from home flexibility
- Sydney-based role
- Small, high-calibre team – no bureaucracy, things ship
- Structured onboarding with increasing technical ownership
- High-trust environment with strong autonomy
- Direct access to founders and real influence over product direction
- Experience using AI tools to improve workflow is encouraged
